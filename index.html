Get Started

Up and running in under 3 minutes

1

Configure MCP Server

Add to ~/.claude.json (macOS/Linux) or %APPDATA%\claude\config.json (Windows):

{
  "mcpServers": {
    "doclea": {
      "command": "npx",
      "args": ["@doclea/mcp"]
    }
  }
}
2

Restart Claude Code

Close and reopen Claude Code to load the new MCP server.

‚úì Doclea will be available in your tools
3

Initialize Your Project

Navigate to your project directory, then ask Claude:

"Initialize doclea for this project"

Expected output:

‚úì Analyzed 45 commits
‚úì Imported 3 existing docs
‚úì Stored 12 initial memories
‚úì Ready for search
